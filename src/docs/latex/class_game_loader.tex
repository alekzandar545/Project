\doxysubsection{Detailed Description}
The \doxylink{class_game_loader}{Game\+Loader} class provides functionality to load and display game data. 

The \doxylink{class_game_loader}{Game\+Loader} class contains static methods to load game data, such as player information, high scores, and game maps, from a save file. It also provides a method to display the high scores.
