\doxysubsection{Detailed Description}
Represents an entity in the game. 

This class provides functionality for managing an entity\textquotesingle{}s attributes such as strength, mana, health points, armor, and dexterity. It also includes methods for taking hits, healing, and accessing the entity\textquotesingle{}s attributes.
