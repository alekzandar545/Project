\doxysubsection{Detailed Description}
Represents the user interface for a battle in the game. 

The \doxylink{class_battle_u_i}{Battle\+UI} class is a derived class of the \doxylink{class_selection_u_i}{Selection\+UI} class and provides methods for rendering graphics, rendering the UI, and rendering HP bars. It also contains a pointer to a \doxylink{class_monster}{Monster} object, as well as vectors for storing player and dragon art.
