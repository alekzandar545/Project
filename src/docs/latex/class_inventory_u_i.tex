\doxysubsection{Detailed Description}
The \doxylink{class_inventory_u_i}{Inventory\+UI} class represents the user interface for displaying and managing the inventory. 

It inherits from the \doxylink{class_selection_u_i}{Selection\+UI} class and provides functionality for rendering the inventory, moving the selection, and checking the selected item.
