\doxysubsection{Detailed Description}
Represents a monster entity in the game. 

The \doxylink{class_monster}{Monster} class is a derived class of the \doxylink{class_entity}{Entity} class and represents a monster entity in the game. It provides functionality for combat, including choosing attacks and performing attacks on other entities.
