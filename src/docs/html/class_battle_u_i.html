Detailed Description

Represents the user interface for a battle in the game.

The BattleUI class is a derived class of the SelectionUI class and provides methods for rendering graphics, rendering the UI, and rendering HP bars. It also contains a pointer to a Monster object, as well as vectors for storing player and dragon art.
